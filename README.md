#  Proyecto: Tienda Web ToteBags - Base de Datos

Este repositorio contiene la estructura y lógica de base de datos para el sistema de gestión de una tienda online.

# Contenido del Proyecto
* **Tablas:** Definición de clientes, productos, diseños, pedidos, metodosPago, visitas, pedido_producto, producto_diseño.
* **Triggers:** Automatización de procesos lógicos.
* **Consultas:** Scripts optimizados para reportes.

# Para ejecutarlo
1. Ejecuta el script maestro: `script.github.sql`.
2. Carga los datos iniciales desde `datosdetablas.sql`.
